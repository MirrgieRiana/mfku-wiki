# ifrku-wiki-skill

IFRKU非公式Wikiの記述規約を提供するClaude用のスキルです。
Wiki記事の編集やレビューの際に参照してください。

## 構成ファイル

- [KNOWHOW.md](scripts/ai/KNOWHOW.md)
   - プロジェクトの概要や資料、教訓などのノウハウが記載されています。
- [SKILL.md](src/main/jekyll/SKILL.md)
   - スキルの概要と各ファイルの説明です。
- [ifrku-wiki-skill.md](src/main/jekyll/ifrku-wiki-skill.md)
   - 記述規約の本体です。
