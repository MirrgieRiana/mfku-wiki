
2023年に開発がスタートしたMirageFairyの3番目の公開MOD。
2025年2月1日時点での著者は、「MirageFairy開発者-G7」。

----

MF24KUの派生元の作品。

MirageFairy2019やMirageFairy2023の後継作であり、舞台は異なるが同じ世界観を共有している。
- 一部の機能もある程度形を変えながら続投している。

* 依存関係 [#b0e39d3d]

- Minecraft 1.20.1
- Fabric
- TerraBlenader
- Fabric Language Kotlin
- Fabric API
- Cloth Config API
- oωo (owo-lib)
- Faux Custom Entity Data
